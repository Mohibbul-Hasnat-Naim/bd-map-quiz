# 🌍 Geo Interactive Learning

An interactive, browser-based geography learning platform that uses **map-based quizzes** to build fast, visual understanding — from **Bangladesh to the world**.

---

## 📌 Overview

**Geo Interactive Learning** is a scalable quiz system where users learn geography by directly interacting with maps.

Instead of memorizing lists, users **click, explore, and recall** locations — making learning faster and more intuitive.

The platform supports both:

* 🇧🇩 **Bangladesh-focused learning** (divisions, districts)
* 🌍 **Global geography** (countries, continents, HQ locations, straits)

---

## 🎯 Features

* 🗺️ Interactive SVG map-based quizzes
* 🌍 Dual scope: **Bangladesh + World**
* 🧠 Multiple learning modes:

  * Map Mode (click on map)
  * Option Mode (MCQ-style)
  * Hybrid Mode (planned)
* 🔍 Zoom & pan for smooth navigation
* 📊 Score tracking & performance feedback
* ⚡ Fully browser-based (no install required)
* 🧩 Modular architecture (easy to expand topics)

## 🧠 Learning Approach

This project is built on:

* **Visual memory (map interaction)**
* **Active recall (quiz system)**
* **Repetition through gameplay**

Result: faster learning + better retention.

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – UI & layout
* **JavaScript (Vanilla)** – Quiz engine & interaction
* **SVG Maps** – Geographic rendering

---

## ⚙️ Architecture (Simplified)

* `quizEngine` → controls logic, scoring, flow
* `mapAdapter` → handles different map types (BD / World)
* Modular page system → easy to add new quizzes
