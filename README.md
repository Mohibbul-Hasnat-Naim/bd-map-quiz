# Bangladesh Map Quiz

An interactive, browser-based educational quiz application designed to help users learn and memorize the districts of Bangladesh through an engaging map interface.

---

## 📌 Overview

**Bangladesh Map Quiz** is a web-based learning tool where users interact with a scalable SVG map of Bangladesh and identify districts. The project focuses on making geography learning simple, visual, and interactive.

It supports both **learning mode** and **quiz mode**, allowing users to practice and test their knowledge in a structured way.

---

## 🎯 Features

- 🗺️ Interactive SVG map of Bangladesh
- 🧠 Quiz mode for testing district knowledge
- 📘 Learning mode with guided exploration
- 🔍 Zoom and pan support for better map navigation
- 🖱️ Hover and click-based interaction
- 📊 Score tracking (correct answers, attempts, progress)
- ⚡ Lightweight and runs directly in the browser (no installation needed)

---

## 🛠️ Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling and responsive layout
- **JavaScript (Vanilla)** – Core logic and interactivity
- **SVG Map Data** – Geographic rendering of Bangladesh
